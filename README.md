# Renegades Add-Ons Russian
Дополнения для Mayhem 3 Renegades, содержащие исправления ошибок и ранее предложенные функции. Перед обновлением игры обязательно ознакомьтесь со списками изменений в папке релизов!

Форум: https://forum.egosoft.com/viewtopic.php?t=473331

Установка:
- Renegades должен быть установлен ДО установки Дополнений. Следуйте инструкциям по установке Renegades здесь: https://forum.egosoft.com/viewtopic.php?t=460221
- После установки Renegades просто скопируйте папки addon и mayhem_data в корневую директорию вашей игры.
- Скопируйте файлы TShips.txt и TBullets.txt из папки ZMap этого проекта в вашу папку ZMap.

Политика управления версиями:
- Минорные версии (например, 1.X) могут изменять поведение постоянного мира и в отдельных случаях могут требовать начала новой игры.
- Патч-версии (например, 1.5.X) безопасны для обновления в процессе сохранённой игры.
